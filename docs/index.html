Mental Models

First Principles Thinking

Analytical

Break down complex problems into fundamental components and build up from basics.

  • Identify the problem clearly
  • Break it down into basic components
  • Question assumptions
  • Build up from the basics

Opportunity Cost Analysis

Decision-making

Consider what you give up when making a choice to make better decisions.

  • Identify all available options
  • List the benefits of each option
  • Identify what you give up with each choice
  • Compare opportunity costs

Bayesian Thinking

Probabilistic

Update beliefs based on new evidence using probabilistic reasoning.

  • Start with prior beliefs
  • Gather new evidence
  • Update beliefs using Bayes' theorem
  • Consider alternative explanations

Systems Thinking

Holistic

Understand how parts of a system interact and consider emergent properties.

  • Identify system boundaries
  • Map system components
  • Identify relationships and feedback loops
  • Consider emergent properties

Custom Mental Models

Create your own mental models with YAML configuration:

models:
  my_custom_model:
    name: "My Custom Model"
    description: "A custom approach to problem solving"
    steps:
      - "Step 1: Define the problem"
      - "Step 2: Gather information"
      - "Step 3: Analyze patterns"
      - "Step 4: Generate solutions"
    category: "custom"
    priority: 10  # Higher = more priority

git clone https://github.com/rainmana/gothink.git

Usage

1

Configure MCP

Add GoThink to your MCP configuration:

{
  "mcpServers": {
    "gothink": {
      "command": "gothink",
      "args": []
    }
  }
}
2

Custom Mental Models (Optional)

Create custom mental models with YAML:

export GOTHINK_MENTAL_MODELS_PATH=/path/to/mental_models.yaml
3

Start Using

Use the mental model tools in your AI system:

{
  "tool": "mental_model",
  "arguments": {
    "session_id": "my-session",
    "model_name": "first_principles",
    "problem": "How to optimize team productivity"
  }
}

API Reference

mental_model

Apply mental models to solve problems using structured thinking frameworks.

Parameters:

  • session_id (required) - Session identifier
  • model_name (required) - Name of the mental model to apply
  • problem (required) - Problem statement to analyze
  • steps (optional) - Custom steps to follow

list_mental_models

List all available mental models with their details and priorities.

Parameters:

  • None

sequential_thinking

Perform sequential thinking operations with structured thought progression.

Parameters:

  • session_id (required) - Session identifier
  • thought (required) - Current thought content
  • thought_number (required) - Current thought number in sequence
  • total_thoughts (required) - Total number of thoughts planned
  • next_thought_needed (required) - Whether another thought is needed
